# Layout Specification — Neo-Forensic Cyberstructuralism

## 1. Page / Canvas
- Aspect ratio: 1.6:1 (ideal EPUB cover ratio)
- Background: #000000 (pure black)
- Safe margins: 8% on all sides

## 2. Grid System
Primary grid:
- 12-column orthographic grid
- 16 baseline rows
- Gutters: 1–2% width

Optional alternate grids:
- Radial grid for theorem-like covers
- Hexagonal computational lattice for cognitive themes

## 3. Title Block
- Font: League Gothic Regular
- Size: 10–14% of total canvas height
- Alignment: centered
- Case: ALL CAPS
- Letter spacing: -2% to -4%
- Color: white (#FFFFFF)
- Position: 5–10% from top

## 4. Subtitle / Metadata
- Font: Lexend Light / Regular
- Size: 3.5–5% of canvas height
- Alignment: centered
- Position: beneath title by 2–3% height
- Content examples:
  - “Report No. 027 | Year 2026”
  - “Dataset 44-B | Revision 2.1”

## 5. Illustration Block
- Position: center of canvas
- Width: ~60–70% of canvas width
- Must remain abstract
- Must use monoline strokes (recommend 2–3px relative)
- Glow: accent color with opacity 40–70%

## 6. Bottom Metadata Strip
- Font: Lexend Regular (small)
- Alignment: centered
- Position: 5–8% from bottom
- Content: plate codes, compilation date, dataset info

## 7. Optional Frame
- Thin inner border
- Inset: 2–4%
- Color: dim accent (20–30% opacity)
