# Neo-Forensic Cyberstructuralism — Visual Identity Guide
Author: Salvador Guzman
Version: 1.0

## 1. Overview
Neo-Forensic Cyberstructuralism is a visual system designed for nonfiction works 
that merge engineering, mathematics, social structure analysis, and human 
complexity. It expresses emotional or historical material only through 
abstract geometry and computational aesthetics.

Covers present themselves as technical plates from a future research archive: 
precise, luminous, analytical, and austere.

## 2. Core Principles
- Black background (absolute requirement)
- Single luminous accent color per cover
- Abstract monoline diagrams (no realism ever)
- Strict grid discipline (orthographic, radial, or hex)
- Research-archive annotations (plate numbers, dataset ids)
- Typography = industrial + legible
- Composition = centered focus, wide margins, balanced geometry

## 3. Color Philosophy
Accent colors represent conceptual “energy modes”:

- Axiom Blue → computational clarity
- Protocol Red → sociotechnical tension
- Signal Orange → structural stress / resilience
- Spectral Violet → cognitive/emotional dynamics
- Archive Gold → theorem-heavy formalism

Strict rule:
**One accent color per book**.  
Never mix modes across a single cover.

## 4. Typography
### Title
- **League Gothic (Regular)**
- All caps
- Tight letter spacing (-2% to -4% depending on renderer)
- Title is allowed to be loud and tall

### Metadata / Body
- **Lexend (Light or Regular)**
- Small sizes acceptable, as it is highly readable
- Used for: report IDs, dataset numbers, dates, specimen codes

## 5. Illustration Style
### Must:
- Use monoline strokes
- Use vector-clean geometry
- Be abstract (network graphs, polyhedral forms, knot diagrams)
- Use glow sparingly around nodes/edges
- Maintain symmetry or structured asymmetry

### Must Not:
- Contain photography
- Contain 3D render shading
- Contain realistic objects (papers, gears, faces)
- Contain gradients except glow halos

## 6. Composition Rules
- Main illustration centered
- Safe margins: 7–10% on all sides
- Title block sits top-center
- Metadata sits bottom-center
- Optional: thin framing line inset 2–4%

## 7. Optional Flourishes
- Scanline noise (very faint)
- Particle sparks around nodes
- Micro-text annotations: “Plate 07-C”, “Revision 2.1”

## 8. Use Cases
- Research Reports
- Mathematical Monographs
- Sociotechnical Analysis Papers
- Historical Analytics
- Engineering Studies
